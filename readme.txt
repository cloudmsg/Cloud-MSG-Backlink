== Description ==

This plugin is a widget that shows a backlink to cloudmsg.com

== Installation ==

How to install the plugin...

1. Upload `cloudmsg_backlink` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
